# 📥 ПОДКАЧКА ФАЙЛОВ — Умный менеджер загрузок

> Профессиональный менеджер загрузки файлов с графическим веб-интерфейсом и продвинутыми возможностями

## 📋 О проекте

**Подкачка файлов** — это мощный инструмент для скачивания файлов из интернета, сочетающий удобный веб-интерфейс и производительную консольную версию. Идеальное решение для тех, кто ценит скорость, надежность и контроль над загрузками.

## 🚀 Ключевые возможности

| Функция | Описание |
|---------|----------|
| **⚡ Параллельные загрузки** | До 3 файлов одновременно (настраивается) |
| **🔄 Возобновление** | Поддержка .part файлов для докачки прерванных загрузок |
| **📊 Прогресс-бар** | Визуальное отображение прогресса через tqdm |
| **🏷️ Умные имена** | Автоматическое определение имен файлов |
| **📑 Без дубликатов** | Автоматическое переименование при совпадении имен |
| **📈 Детальные отчеты** | TXT и JSON отчеты на рабочем столе |
| **🌐 Веб-интерфейс** | Красивый UI на Flask с управлением через браузер |
| **⚙️ Гибкие настройки** | Конфигурация через JSON файл |
| **📝 Логирование** | Полный лог прогресса загрузок |

```bash
# Запустите файл

# Или через Python
python web_interface.py

Следите за прогрессом в реальном времени!

💻 Консольная версия
Отредактируйте urls.txt, добавьте свои URL

Запустите:

bash
python downloader.py
📍 Результаты загрузки
📁 Файлы сохраняются в папку Загрузки/

📄 Отчет на рабочем столе: download_report.txt

📊 Детальный JSON отчет: download_detailed_report.json

⚙️ Настройки (settings.json)
json
{
  "download_folder": "Загрузки",      // Папка для сохранения
  "max_concurrent": 3,                // Одновременных загрузок
  "timeout": 30,                      // Таймаут (секунд)
  "chunk_size": 8192,                 // Размер чанка (байт)
  "resume": true,                      // Возобновлять загрузки
  "user_agent": "Mozilla/5.0...",      // User-Agent
  "retry_count": 3,                    // Попыток при ошибке
  "retry_delay": 2                      // Задержка между попытками
}
🌐 Веб-интерфейс - Функции
📊 Статистика	Задачи, завершено, ошибки в реальном времени

📁 Файловый менеджер	Просмотр загруженных файлов
📄 Отчеты	Скачивание подробных отчетов
⚙️ Настройки	Просмотр текущей конфигурации
🔄 Автообновление	Статус обновляется автоматически

⚠️ Ограничения
❌ Не поддерживает FTP

❌ Не работает с авторизацией (логин/пароль)

❌ Может сталкиваться с Cloudflare защитой

⏱️ Максимальный таймаут: 30 секунд (настраивается)

💡 Советы
Для больших файлов — увеличьте timeout в settings.json

Для множества файлов — установите max_concurrent: 5-10

Файлы .part — временные, удаляются после завершения

При зависании — проверьте download.log в папке Загрузки

Для Яндекс.Диска/Google Drive — нужны прямые URL
