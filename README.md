ПАПКА "ПОДКАЧКА" - УЛУЧШЕННЫЙ МЕНЕДЖЕР ЗАГРУЗКИ ФАЙЛОВ
=======================================================

Эта папка содержит профессиональный менеджер загрузки файлов с графическим веб-интерфейсом.

СТРУКТУРА ПАПКИ:
-----------------
Подкачка/
├── downloader.py           - Консольная версия загрузчика (advanced)
├── web_interface.py        - Веб-интерфейс с красивым UI (РЕКОМЕНДУЕТСЯ)
├── start_web_interface.bat - Быстрый запуск веб-интерфейса
├── urls.txt                - Список URL-адресов для скачивания
├── settings.json           - Настройки загрузчика (редактируйте при необходимости)
├── README.txt              - Этот файл
└── Загрузки/               - Папка для скачанных файлов (создается автоматически)

ОСНОВНЫЕ ФУНКЦИИ:
-----------------
✅ Параллельные загрузки (до 3 файлов одновременно)
✅ Возобновление прерванных загрузок (.part файлы)
✅ Визуальный прогресс-бар (tqdm)
✅ Автоматическое определение имен файлов
✅ Обработка дубликатов (автоматическое переименование)
✅ Подробные отчеты (TXT + JSON)
✅ Веб-интерфейс (Flask) с управлением через браузер
✅ Конфигурация через JSON
✅ Логирование прогресса

БЫСТРЫЙ СТАРТ (ВЕБ-ИНТЕРФЕЙС):
------------------------------
1. Запустите start_web_interface.bat двойным кликом
2. Откройте браузер и перейдите по адресу: http://localhost:5000
3. Введите URL файлов в текстовое поле (по одному на строку)
4. Нажмите "Загрузить файлы"
5. Следите за прогрессом в реальном времени!

ИЛИ используйте консольную версию:
----------------------------------
1. Отредактируйте urls.txt, добавьте свои URL
2. Запустите downloader.py двойным кликом или через cmd:
   python downloader.py
3. Результат будет:
   - В папке "Загрузки"
   - Отчет на рабочем столе: download_report.txt
   - Детальный JSON отчет: download_detailed_report.json

НАСТРОЙКИ (settings.json):
--------------------------
{
  "download_folder": "Загрузки",      - Папка для файлов
  "max_concurrent": 3,                - Одновременных загрузок
  "timeout": 30,                      - Таймаут (секунд)
  "chunk_size": 8192,                 - Размер чанка (байт)
  "resume": true,                     - Возобновлять прерванные
  "user_agent": "...",                - User-Agent для запросов
  "retry_count": 3,                   - Попыток при ошибке
  "retry_delay": 2,                   - Задержка между попытками
}

ТРЕБОВАНИЯ:
-----------
- Python 3.8+
- Библиотеки: requests, flask, tqdm, beautifulsoup4
  Установка: pip install requests flask tqdm beautifulsoup4

ОТЛАДКА:
--------
1. Логи загрузок: Загрузки/progress.log
2. Консольный вывод: При запуске downloader.py
3. Веб-интерфейс: http://localhost:5000 (показывает живую статистику)
4. Отчеты: download_report.txt и download_detailed_report.json на рабочем столе

ОГРАНИЧЕНИЯ:
------------
- Не поддерживает FTP
- Не работает с авторизацией (логин/пароль)
- Может сталкиваться с Cloudflare защитой
- Максимальный таймаут: 30 секунд (настраивается)

СОВЕТЫ:
-------
1. Для больших файлов увеличьте timeout в settings.json
2. Если нужно скачать много файлов - используйте max_concurrent: 5-10
3. Файлы .part - временные, удаляются после завершения
4. При зависании - проверьте download.log в папке Загрузки
5. Для скачивания с Яндекс.Диска/Google Drive нужны прямые URL
